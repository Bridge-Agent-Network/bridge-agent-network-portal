# Bridge Agent Network Dashboard Content Checklist

Use this checklist to decide what content, workflows, and business rules are needed to turn the dashboard from a polished MVP into a working Bridge Agent Network portal.

## Core Positioning

Bridge Agent Network should position each member as a **Seller Options Strategist**.

The portal should help agents guide sellers through every serious path before they decide how to sell:

- Cash Offer
- As-Is Listing
- Reno-to-Sell
- Builder / Spec Home Solution

Recommended core language:

> **Before you sell, know every path.**

Recommended agent-facing language:

> **Become the agent sellers call before they decide how to sell.**

Recommended seller-facing language:

> **Know every path before you sell.**

## Dashboard Section: Bridge Points

Purpose:

Reward behavior that grows the network, improves seller conversations, and creates deal flow.

Needed decisions:

- Final point values
- What activities require admin approval
- What activities are automatic
- Whether points are public on a leaderboard from day one
- Whether points reset monthly, quarterly, or never
- What rewards points unlock

Recommended point categories:

### Education

- Complete onboarding
- Complete Cash Offer module
- Complete Reno-to-Sell module
- Complete Builder Solution module
- Attend monthly mastermind
- Pass certification refresh

### Activity

- Submit seller opportunity
- Submit land opportunity
- Submit property photos
- Upload seller situation notes
- Use seller options calculator
- Send Bridge presentation to a seller

### Marketing

- Post Bridge-approved content
- Send Bridge email to database
- Mail Bridge postcard campaign
- Host seller options event
- Record seller education video

### Results

- Cash offer accepted
- Reno-to-Sell project launched
- Builder/spec solution advanced
- Closed transaction
- Seller testimonial collected

### Network Growth

- Refer accepted agent
- Invite lender/title/vendor partner to event
- Bring brokerage leadership into presentation

Recommended rewards:

- Priority lead routing
- Featured directory placement
- Custom marketing assets
- Elite badge/status
- Mastermind access
- Agent spotlight
- First look at landowner opportunities

## Core Workflow: Opportunity / Deal Desk

This should likely become the most important next feature.

Purpose:

Allow agents to submit real seller situations and receive strategic guidance from Bridge.

Recommended stages:

1. New Opportunity
2. Seller Discovery
3. Property Review
4. Options Built
5. Seller Presentation
6. Option Selected
7. In Progress
8. Closed / Lost

Needed decisions:

- Who reviews submissions?
- What is the expected response time?
- Do agents need to include seller contact info?
- Can agents save drafts?
- Should admins assign opportunities to Brad/Alex/team members?
- Should opportunity updates trigger notifications?

## Core Tool: Seller Options Calculator

Purpose:

Give agents a simple, visual way to compare seller paths.

The calculator should compare:

- Cash Offer
- As-Is Listing
- Reno-to-Sell
- Builder / Spec Solution

Needed inputs:

- As-is estimated value
- Cash offer estimate
- Estimated list price
- Estimated renovated value
- Renovation budget
- Holding cost estimate
- Commission estimate
- Seller closing cost estimate
- Timeline
- Property condition
- Land/build opportunity notes

Needed decisions:

- Default cash offer discount range
- Default commission assumption
- Default seller closing cost assumption
- Default renovation timeline
- Whether results are agent-only
- Whether results can export as a seller-facing PDF

Recommended output:

**Property Pathway Report**

This report should summarize:

- Seller goals
- Property condition
- Cash offer path
- As-is listing path
- Reno-to-Sell path
- Builder/spec path
- Estimated net comparison
- Recommended next step

## Core Identity: Certification / Status

Purpose:

Make the network feel selected, serious, and professional even though agents do not pay a fee.

Recommended levels:

- Bridge Member
- Bridge Certified
- Bridge Strategist
- Bridge Elite
- Territory Leader

Needed decisions:

- What unlocks each level?
- Does certification require a quiz?
- Does certification require a mock seller presentation?
- Can agents use the badge before certification?
- Does certification expire?
- What makes an agent inactive?

Recommended activation requirements:

- Attend onboarding
- Complete certification
- Submit one seller opportunity every 60 or 90 days
- Use the Bridge seller options presentation
- Share one Bridge marketing piece per month
- Maintain profile accuracy
- Follow seller consultation standards

## Bridge Agent Standards

Purpose:

Make the network feel legitimate and protect seller trust.

Recommended standards:

Bridge Agents commit to:

- Presenting multiple seller pathways honestly
- Never pressuring sellers into one option
- Explaining net, timeline, risk, and convenience
- Protecting the seller’s agency relationship
- Using Bridge tools transparently
- Maintaining professionalism with investors, builders, and vendors
- Keeping profile and territory information accurate
- Participating in ongoing training

Needed decision:

Should agents electronically agree to standards before earning certification?
